👕 Data Augmentation for Fashion MNIST

A Deep Learning project demonstrating the implementation of **Data Augmentation** techniques and **Convolutional Neural Networks (CNNs)** for image classification using the Fashion MNIST dataset. The project focuses on improving model generalization and classification accuracy through augmented image generation and advanced training techniques.

---

🚀 Features
- Image classification using CNN
- Data augmentation for improved model performance
- Fashion MNIST dataset implementation
- Deep Learning workflow with TensorFlow/Keras
- Model training and evaluation
- Visualization of predictions and performance metrics

---

🛠️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook

---

📊 Dataset
The project uses the **Fashion MNIST Dataset**, containing grayscale images of fashion products categorized into 10 different classes such as:
- T-Shirts
- Shoes
- Bags
- Dresses
- Sneakers
- Coats

---

📈 Data Augmentation Techniques
The following augmentation methods are used to enhance dataset diversity:
- Rotation
- Zooming
- Width & Height Shifting
- Horizontal Flipping
- Rescaling

These techniques help reduce overfitting and improve model robustness.

---

🎯 Objective
The primary goal of this project is to develop a high-performing image classification model while demonstrating the effectiveness of data augmentation in Deep Learning applications.
